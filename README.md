# SEF
Subdomain Enumeration Framework

# Requirnment 
* Subenum
    * Subfinder
    * Findomain
    * Assetfinder
    * Amass
    * Anew
* Sub.sh
    * Parallel
    * JQ
    * Httprobe
* ShuffleDNs
    * MassDns
    * all.txt
    * resolver.txt
* Dnsgen
    * word.txt
* Httpx

# Usage

### Without Permute

```
SEF0 site.com
```
### With Permute

```
SEF site.com
```

### With Double Permute

```
SEF2 site.com
```
